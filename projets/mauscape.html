Mauscape.

Jeu d'aventure textuel de type "escape game" jouable entièrement dans le terminal. Création d'un moteur de jeu textuel, gestion de scénarios à embranchements et plusieurs fins possibles selon les choix du joueur.

10

Mauscape
Terminal RPG.

Java · Terminal · Game Design

Le contexte.

Premier projet de jeu complet, réalisé en début de formation BUT Informatique. Objectif : créer une expérience interactive textuelle riche, avec une vraie écriture, des choix qui comptent, et un moteur extensible.

La stack.

LangageJava
InterfaceTerminal (CLI)
ApprocheGame design narratif

Les défis.

  • Construire un moteur de jeu textuel extensible sans bibliothèque tierce. Solution : architecture en arbre de scènes, chaque scène pointant vers ses suites possibles selon les choix.
  • Gérer plusieurs fins selon les choix du joueur. Solution : système de flags et de conditions sur les transitions.
  • Maintenir la cohérence narrative avec plusieurs branches de scénario. Solution : carte de scénario écrite en parallèle du code.

Les apprentissages.

Logique de programmation fondamentale, gestion des entrées / sorties, structuration de code en début de parcours, premier projet complet de jeu — c'est ce projet qui a renforcé ma passion pour le développement logiciel.
