Jeu Risk.

Implémentation complète du célèbre jeu de stratégie Risk en Java exécutable dans le terminal. Mode solo contre IA et multijoueur local, architecture orientée objet avec design patterns classiques.

07

Risk
Java OOP.

Java · POO · Terminal · Algorithmique

Le contexte.

Projet académique BUT Informatique. Objectif : implémenter un jeu de stratégie complexe (Risk) avec règles fidèles, gestion de plusieurs joueurs (humains + IA), et boucle de jeu robuste exécutable en ligne de commande.

La stack.

LangageJava
ApprochePOO + design patterns
InterfaceTerminal (CLI)

Les défis.

  • Implémenter la logique stratégique de l'IA pour les adversaires non-humains. Solution : algorithmes de décision basés sur l'évaluation de positions (forces relatives, frontières, menaces).
  • Gérer un état de jeu complexe (territoires, troupes, cartes, tours de jeu). Solution : modèle objet riche avec encapsulation stricte et invariants vérifiés à chaque transition.
  • Boucle de jeu robuste avec tour-par-tour et phases multiples (renforcement, attaque, déplacement). Solution : pattern State pour les phases.

Les apprentissages.

Programmation orientée objet avancée, algorithmes de stratégie pour IA, gestion de la mémoire en Java, application de design patterns (State, Strategy, Observer), maîtrise de la modélisation d'un domaine métier riche.
